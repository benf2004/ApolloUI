Instructions:

  1. Go to the Reddit app creation page.
  2. Click on "create an app"
  3. Name it whatever you want
  4. Select "web app"
  5. Write a description "doing research" or "testing an app" etc
  6. Put an about URL (doesn't really matter)
  7. Copy the redirect URI from above into the redirect URI box (these HAVE to match)
  8. Click "create app"
  9. Paste the string of text under "web app" into the Client ID box.
  10. Paste the random string of text next to "secret" into the Client Secret box
  11. Make a user-agent (this is what reddit will identify your app as)
  12. Click "authorize", sign in to reddit, and you're good to go!

Note: we do not condone breaking the Reddit API terms. You should only use this for noncommercial or research purposes.
